# Fetching Episode Images

To get actual episode thumbnails from the shows, you can:

1. **Use TMDB API** (requires free API key):
   - Sign up at https://www.themoviedb.org/
   - Get API key
   - Use endpoints like: `https://api.themoviedb.org/3/tv/{tv_id}/season/{season_number}/episode/{episode_number}?api_key={api_key}`
   - Image URLs: `https://image.tmdb.org/t/p/w400/{still_path}`

2. **Manual URLs from public sources**:
   - IMDb episode galleries
   - Fancaps.net
   - SpoilerTV

For now, we'll use placeholder URLs that you can replace with actual episode images.
